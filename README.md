# MapMentor - Find Real Tutors Near You

Live: https://mapmentor.vercel.app

Hyderabad's local tutor marketplace. 500+ verified tutors in Gachibowli, Madhapur, Kondapur. No commission.

## Problem
Parents in Hyderabad search WhatsApp groups & Justdial for tutors. Fake profiles, no location filter, 30% commission.

## Solution
MapMentor: Area-wise search, verified tutors, direct contact, ₹0 commission.

## Features
- 🔍 Search by subject + area + price
- 📍 Distance in km (Gachibowli, Madhapur, etc)
- ⭐ Ratings + experience + subjects
- 💬 Direct contact - no middleman
- 📱 Mobile-first, <2 sec load

## Tech Stack
- Next.js 15 (App Router)
- Supabase (DB + Auth)
- Tailwind CSS
- Vercel Deploy
- next/image optimization

## Areas Covered
Gachibowli, Hitech City, Madhapur, Kondapur, Kukatpally, Ameerpet, Secunderabad, Dilsukhnagar

## Subjects
CBSE Math, JEE Math, NEET Biology, Physics, Chemistry, Spoken English, Coding, UPSC
