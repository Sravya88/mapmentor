## MapMentor - Find Real Tutors Near You

Live: https://mapmentor.vercel.app

**Stack:** Next.js 14, Supabase, Tailwind, Vercel

**Features:**
- Search mentors by area/subject with Supabase real-time DB
- Admin dashboard with Auth + image upload to Supabase Storage
- SEO: dynamic sitemap, area landing pages (/tutors-in/Gachibowli)
- Fully responsive, RLS secured

**Why I built:** Hyderabad has no hyperlocal tutor discovery. Justdial/UrbanCompany are generic.
